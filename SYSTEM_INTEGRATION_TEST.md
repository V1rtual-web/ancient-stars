# 系统集成测试清单

## 测试概述

本文档用于验证旧星背单词学习管理系统的完整功能和集成情况。

## 测试环境

- **后端**: Spring Boot + MySQL
- **前端**: Vue 3 + Element Plus
- **浏览器**: Chrome, Firefox, Safari, Edge
