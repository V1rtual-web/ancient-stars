# 响应式设计测试文档

## 测试概述

本文档用于测试旧星背单词学习管理系统前端的响应式设计实现。

## 测试环境

### 屏幕尺寸断点

- **移动端**: < 768px
- **平板端**: 768px - 1024px
- **桌面端**: > 1024px

### 测试设备

- 移动端: iPhone SE (375px), iPhone 12 Pro (390px), Samsung Galaxy S20 (360px)
- 平板端: iPad (768px), iPad Pro (1024px)
- 桌面端: 1280px, 1440px, 1920px

## 测试方法

### 1. Chrome DevTools测试

```
1. 打开Chrome浏览器
2. 按F12打开开发者工具
3. 点击设备工具栏图标（Ctrl+Shift+M）
4. 选择不同设备进行测试
5. 测试横屏和竖屏模式
```

### 2. 真机测试

```
1. 在本地网络中启动开发服务器
```
