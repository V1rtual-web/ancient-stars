# 系统集成测试清单

## 测试概述

本文档列出了旧星背单词学习管理系统的完整集成测试清单，包括后端API测试、前端功能测试、前后端集成测试和跨浏览器兼容性测试。

**测试日期:** 2024年
**测试范围:** 教师端 + 学生端完整功能
**测试环境:**

- 后端: Spring Boot (端口: 8080, 上下文路径: /api)
- 前端: Vue 3 + Vite (开发服务器)
- 数据库: MySQL 8.0 (数据库名: ancient_stars)
